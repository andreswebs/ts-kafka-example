# ts-kafka-example

Example KafkaJS usage in TypeScript.

## Configuration

Add the broker hostnames to `src/config.ts` before running.

## Run

```sh
ts-node ./src/producer.ts
```

```sh
ts-node ./src/consumer.ts
```
